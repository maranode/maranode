Import a model

Models must be in GGUF format. Import from a local file using the CLI: 

# Language model (default)
maranode model import /path/to/model.gguf --name mymodel --tag latest

# Embedding model
maranode model import /path/to/bge-m3.gguf --name bge-m3 --tag latest --type embedding
